## 5. 关键发现

1. **`<Handle>` 必须显式声明** —— 自定义节点没有 `<Handle type="source"/target">` 时，边渲染数 = 0。修复方案：在 ScreenNode 内 `<Handle position="left" type="target" />` + `<Handle position="right" type="source" />`，可设 `opacity: 0` 隐藏视觉
2. **`vite-plugin-singlefile` 配置要求**：必须设 `build.assetsInlineLimit: 100000000` + `build.rollupOptions.output.inlineDynamicImports: true` + `cssCodeSplit: false`，否则可能产出 hash 文件
3. **图片内联代价可控**：测试中 base64 占位图体积忽略不计；18 张真实 WebP（每张约 200KB）内联后单文件预估 **4–5MB**，邮件附件传递无问题
4. **React Flow `<Controls />` 内置完整工具栏**（缩放/Fit/锁定），不必自写
